Приложение для постепенного отказа от курения

Хранения данных в localStorage, пока что без интеграции с БД, без авторизации

Функционал
- mobile first design, пользоваться в основном будут с мобильного телефона
- Возможность фиксировать время каждой выкуренной сигареты
- Видеть сколько времени прошло от прошлой сигареты(таймер)
- Статистика по дням
