# Несколько режимов работы приложения

На главном экране сверху должен быть выпадающий список позволяющий переключить режим

**История сигарет общая для обоих режимов**

---

## Режим "Наблюдение"

В данном режиме человек будет отслеживать сколько сигарет в день он выкуривает

Функционал на главной странице:
- Таймер, считает время от последней сигареты
- Кнопка "Выкурил сигарету"
- Кнопка "Добавить вручную"
- Виджет, показывающий записи о пяти последних сигаретах с указанием времени

---

## Режим "Сокращение"

С помощью данного режима человек будет постепенно сокращать количество сигарет

### Начальное состояние (до настройки)

- Кнопка "Настроить программу", при нажатии появляется модалка

### Модалка настройки программы

Поля:
- Сколько сигарет в день выкуриваете сейчас
- До скольки сигарет хотите сократить
- Период сокращения (дней)
- Часы бодрствования (например 16, если спите 8 часов)
- Тип сокращения (выбор):
  - **Линейное** — плавное уменьшение каждый день
  - **Ступенчатое** — уменьшение на 1 сигарету каждые N дней

### После настройки — внешний вид главного экрана

- **Индикатор дня программы** — "День 7 из 30" + прогресс-бар
- **Таймер обратного отсчёта** — считает время до следующей сигареты
  - Когда таймер достигает 0, показывает "Можно курить!"
- **Прогресс дня** — "Выкурено 5 из 15 разрешённых" + прогресс-бар
- Кнопка "Выкурил сигарету"
- Кнопка "Добавить вручную"
- Виджет, показывающий записи о пяти последних сигаретах с указанием времени

### Логика расчёта

**Интервал между сигаретами:**
```
интервал = (часы_бодрствования * 60) / дневной_лимит минут
```

**Линейное сокращение:**
```
дневной_лимит = текущие - ((текущие - цель) / всего_дней * номер_дня)
```

**Ступенчатое сокращение:**
```
дней_на_шаг = всего_дней / (текущие - цель)
шагов_пройдено = номер_дня / дней_на_шаг
дневной_лимит = текущие - шагов_пройдено
```

**Если выкурил раньше таймера:**
- Пересчитать интервал для следующей сигареты

### В настройках приложения

- Кнопка "Сбросить программу" для сброса настроек режима сокращения
